POST 04 buy insurance
{{base_url}}/omnic/api/insuranceService/insurance/policy/new

buy insurance function is used to purchase insurance coverage.

Required:

Run the get subrisks endpoint first. This step is necessary to retrieve the available options for the insuranceClass. Once the subrisk options are retrieved, select the appropriate insuranceClass value and include it in the payload for the subsequent request.

Sample code: insuranceClass: “PRIVATE COMPREHENSIVE”

Validation Rules:

  • Date: The date should be in day/month/year format (e.g., 12/01/2025).

  • Car Registration No.: The car registration number must not be greater than 11 characters.

  • Chasis Number: The chasis number must not contain the letters “I” and “O”.

The length of the chasis number must not be more than 17 characters.

  • Sum Insured Type: The sum insured should be numeric (i.e., a number)

GET 08. get car model by id
{{base_url}}/omnic/api/insuranceService/insurance/carmodel/48

This endpoint retrieves information about a specific car model based on the provided car model ID.

It returns the car model’s characteristics, such as make, year, engine type, and other specifications.

REQUIRED PARAMETER:
- model ID: The unique identifier for the car model you want to retrieve.

EXAMPLE RESPONSE(json)

{
“modelId”: 123,
“make”: “Toyota”,
“model”: “Corolla”,
“year”: 2020,
“engineType”: “Hybrid”,
“color”: “Blue”,
“price”: 25000
}

Example Request
  • 08. get car model by id
